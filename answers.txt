Q1: Why might we consider using Git?

Answer: We may consider using Git because of the ability to collaborate with others using a branch based system. Synchronized, team based work is the key to success.

Q3: What is the difference between Merging and Rebasing?

Answer: Merging combines the changes from one branch to another, whereas rebasing rewrites the commit history by moving the commits of one branch onto another.
